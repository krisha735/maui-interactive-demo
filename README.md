## 🚀 Project Overview
This project is a hands-on exploration of **.NET MAUI (Multi-platform App UI)**, a modern framework for building cross-platform applications using a single codebase.

The application demonstrates fundamental UI components, event handling, and customization techniques across platforms like:

* Android
* iOS
* Windows
* macOS

---

## 🎯 Objectives
* Understand core UI components in .NET MAUI
* Implement event-driven programming
* Customize UI elements for better user experience
* Build a functional cross-platform application
---
## 🧩 Features Implemented

### 1. 🏷️ Label Display

* Basic label rendering on screen
* Displays static text

### 2. 🎨 Label Customization

* Modified:

  * Text color
  * Font size
  * Alignment

### 3. ✨ Formatted Text

* Multiple styles in a single label
* Different fonts and colors applied

### 4. 🔘 Button Integration

* Added interactive button component

### 5. ⚡ Button Click Event

* Button triggers:

  * Text rotation (180°)
* Demonstrates event handling

### 6. 🎯 Button Styling

* Customized:

  * Background color
  * Border
  * Text appearance

### 7. ⌨️ Entry Field

* User input field added
* Displays typed text dynamically

### 8. 🔄 Text Change Handling

* Converts input text to **UPPERCASE** on Enter
* Demonstrates real-time event response

### 9. 🔢 Input Validation

* Accepts **numeric values only**
* Displays error message for invalid input
* Resets field when incorrect data is entered

### 10. 📝 Editor (Multi-line Input)

* Allows multi-line text input
* Displays entered text using alert on button click
